# Katha Connect

A modern platform designed to connect volunteers with NGOs through an accessible and user-friendly digital experience.

## Features
- Volunteer discovery system
- NGO connection workflows
- Modern responsive UI
- User-focused interaction design

## Tech Stack
- TypeScript
- React
- Tailwind CSS
- Supabase

## Overview
Katha Connect was created to simplify the process of connecting volunteers with organizations working toward meaningful social impact. The project focuses on accessibility, usability, and modern digital interaction patterns.

## Status
Currently improving and expanding features.
